# Installation Notes:
# 1. Pandoc must be installed separately on the system
#    - Windows: Download from https://pandoc.org/installing.html
#    - Ubuntu/Debian: sudo apt-get install pandoc
#    - macOS: brew install pandoc
#
# 2. Tesseract OCR must be installed for OCR functionality
#    - Windows: Download from https://github.com/UB-Mannheim/tesseract/wiki
#    - Ubuntu/Debian: sudo apt-get install tesseract-ocr
#    - macOS: brew install tesseract
#
# 3. Additional Tesseract language packs may be needed
#    - Ubuntu/Debian: sudo apt-get install tesseract-ocr-[lang]
#    - Windows: Download language packs from GitHub
#
# 4. LibreOffice may be required for advanced ODT support
#    - This provides better compatibility with OpenDocument formats
